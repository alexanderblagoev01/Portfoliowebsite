# Bogota Restaurant — Website UI Kit

A high-fidelity recreation of the Ресторант Богота landing page, with reusable components for hero, menu, reservation, and footer surfaces.

## Files

- `index.html` — interactive landing page demo. Click "Запазете маса" to see the reservation flow.
- `Header.jsx` — top nav with dual-script logo and primary CTA.
- `Hero.jsx` — full-bleed hero with script eyebrow + display heading.
- `About.jsx` — centered editorial section about the restaurant.
- `MenuSection.jsx` — three-column menu grid with category headings.
- `ReservationModal.jsx` — modal reservation form, three steps.
- `Footer.jsx` — dark footer with hours, contact, address.

## Notes

- All copy is Bulgarian (Cyrillic). English equivalents live in code comments.
- Photography is represented by colored placeholder blocks with food-evoking warm tones; swap in real photography when available.
- Reservations are mocked — submitting renders a success card after a 600ms simulated post.
